Bulk Ordering

Bulk Order Processing extension facilitates the admin to process multiple orders at a time, without having to visit each order page, individually. In a single orders page, the admin will find a list of orders placed by the customers. It is feasible for the admin to create invoices, generate shipments, add tracking numbers, change order status for multiple orders in the order page, in their respective columns.
